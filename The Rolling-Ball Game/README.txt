Description:

This program is a shooting game, but is not completed. It consist
of two players, where the two players have a particular ball and 
they play against each other, shooting their balls to destroy the 
opponent balls. The balls are distributed randomly inside a bounded 
area with another 3 type of balls, one type is red that represents 
a bomb, another black which represent a fixed ball, and a yellow 
happy face ball that represents a bonus ball. Also the program has 
login feature, and some others features as history, play new game
how to play that was not completed.
